## ğŸ› ï¸ Troubleshooting

### Login Issues

**Problem:** Can't log in  
**Solution:**

1. Check email is correct
2. Reset password via "Forgot Password"
3. Clear browser cache
4. Try incognito/private mode

### Pairing Issues

**Problem:** QR code won't scan  
**Solution:**

1. Ensure client has GardenBuddy app downloaded
2. Check phone camera has permission
3. Try generating new QR code
4. Use shareable link instead (alternative)

**Problem:** Client paired but doesn't see jobs  
**Solution:**

1. Check client's app is connected to internet
2. Wait 60 seconds (sync interval)
3. Client should refresh app
4. Check job was created successfully

### Sync Issues

**Problem:** Jobs not appearing in client app  
**Solution:**

1. Verify pairing is active (check client list)
2. Check internet connection
3. Wait up to 60 seconds for sync
4. Contact support if still broken

### Billing Issues

**Problem:** Trial expired, can't upgrade  
**Solution:**

1. Click "Upgrade" in settings
2. Choose tier
3. Enter payment details
4. Confirm subscription

**Problem:** Payment declined  
**Solution:**

1. Check card details are correct
2. Ensure sufficient funds
3. Try different card
4. Contact your bank
